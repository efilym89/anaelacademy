# Академия Анаэль — Telegram Mini App (демо)

Демо-приложение для внутренней академии компании: курсы, база материалов и профиль сотрудника.

## Что реализовано

- Single-page интерфейс в стиле Telegram Mini App.
- Нижняя навигация с активной вкладкой (Главная / Курсы / Материалы / Профиль).
- Интерактивные карточки: на любой активный блок можно нажать и открыть детальную страницу материала.
- Базовые знания по теме депиляции и сервиса с ссылками на открытые источники (AAD, NHS, CDC).
- Базовая интеграция с Telegram WebApp API (`ready`, `expand`, имя пользователя в шапке).
- Анимации нажатия и плавное появление экранов.

## Локальный запуск

```bash
python3 -m http.server 4173
```

Откройте: `http://localhost:4173`
